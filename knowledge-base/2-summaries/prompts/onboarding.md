# Onboarding Prompts | v1.0.0 | 2026-02-03

Prompt templates used by Resto during the onboarding flow to guide new users.

**Template Gallery Phase:** System prompt that positions Resto as a helpful guide, showing template options and asking what resonates with the user's vision.

**Guided Wizard Phase:** Structured question prompts covering: application type (web/mobile), industry, problem to solve, suggested solution, business model. Each question builds on previous answers.

**Conversational Deep-Dive Phase:** Open-ended exploration prompts. Resto asks nuanced questions about market, competition, differentiation, target users, revenue model, and technical requirements. Adapts based on answers.

**Account Connection Phase:** Step-by-step prompts guiding users through connecting third-party accounts. Explains why each service is needed, provides OAuth flow guidance.

**Plan Generation Phase:** After collecting all inputs, Resto generates the comprehensive business plan and persistent checklist. Presents it for user review and approval before proceeding.
