# User Preferences & Persona | v1.0.0 | 2026-02-03

Each user can customize their AI co-founder experience to match their working style.

**Default Persona — Resto:**
- Branded ATR persona with consistent voice
- Friendly, professional, proactive
- Adapts communication style based on context (shorter in voice, detailed in text)
- Every user gets Resto by default on day one

**Customizable Settings:**
- Name: Users can rename their co-founder
- Personality traits: Formal ↔ Casual, Cautious ↔ Bold, Concise ↔ Detailed
- Communication preferences: Text-first or voice-first
- Notification frequency: Real-time, batched, or minimal
- Working hours: When the agent should proactively reach out

**Autonomy Levels (User-Configurable):**
- Per action category: auto-approve, quick-confirm, or explicit-approval
- Starting point: user chooses during onboarding
- Goal: progressively expand toward full autonomy as trust builds
- Categories: research, code changes, deployments, spending, external communications

**Language & Locale:**
- Multi-language support (priority: English, Portuguese, Spanish)
- Timezone-aware scheduling and notifications
