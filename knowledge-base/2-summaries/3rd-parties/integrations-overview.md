# Third-Party Integrations Overview | v1.0.0 | 2026-02-03

Resto connects to customer-owned third-party accounts to operate their business. All accounts are owned by the customer, operated by the agent with granted permissions.

**Development:** GitHub, GitLab, Bitbucket, Cursor Cloud, Claude Code, Codex
**Hosting:** Vercel, AWS, GCP, Cloudflare, Netlify
**Payments:** Stripe, PayPal, Square
**Communication:** Gmail, Outlook, Slack, Discord, Twilio
**Domains:** Namecheap, GoDaddy, Cloudflare Registrar
**Marketing:** Mailchimp, HubSpot, ConvertKit, social media APIs
**Analytics:** Google Analytics, Mixpanel, PostHog, Amplitude
**CRM:** HubSpot, Salesforce, Pipedrive
**Finance:** QuickBooks, Xero, Stripe Billing
**Legal:** DocuSign, PandaDoc
**Support:** Intercom, Zendesk, Crisp
**Design:** Figma, Canva

**Integration Pattern:** OAuth preferred. API keys as fallback. MCP protocol where supported. All credentials encrypted, audit-logged.

**Priority for V1:** GitHub, Vercel, Stripe, Gmail, Google Analytics, one domain registrar.
